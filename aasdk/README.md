# 文档概述

AASDK-账户SDK，默认集成了如Google，Facebook，Apple，Avidly等主流的登录方式，以及游客登录方式，减少开发者单独对接每一个登录方式的成本，简单快捷。旨在帮助用户在其游戏中快速接入用户登录功能，提升开发效率和用户体验。
